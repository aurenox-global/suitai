## 🇪🇸 Presentación

### ¿Qué hace SuitAI PRO?
SuitAI PRO integra asistentes de texto, voz, documentos e imagen en una experiencia local y privada.

### Capacidades principales
- Chat y asistencia para redacción, programación y productividad.
- Generación y mejora de código con ayuda contextual.
- RAG documental para consultar archivos y extraer conocimiento.
- Voz integrada: síntesis (TTS) y reconocimiento (STT).
- Herramientas de imagen con generación y análisis.
- Entorno con enfoque offline usando modelos locales GGUF.

### ¿Para quién está pensado?
- Desarrolladores que quieren IA local para programar.
- Equipos que necesitan mayor control de datos y privacidad.
- Creadores que combinan texto, voz, documentos e imágenes en su flujo de trabajo.

### Valor diferencial
SuitAI PRO centraliza múltiples capacidades de IA en una sola aplicación, reduciendo fricción entre herramientas y manteniendo el control local del entorno de trabajo.

## 🇬🇧 Presentation

### What does SuitAI PRO do?
SuitAI PRO brings text, voice, document, and image AI features into a private, local-first experience.

### Core capabilities
- Chat assistance for writing, coding, and daily productivity.
- Code generation and improvement with contextual support.
- Document RAG to query files and extract useful knowledge.
- Integrated voice features: text-to-speech (TTS) and speech-to-text (STT).
- Image tools for generation and analysis.
- Offline-oriented workflow powered by local GGUF models.

### Who is it for?
- Developers who want local AI support for coding.
- Teams that need stronger data control and privacy.
- Creators who combine text, voice, documents, and images in one workflow.

### Differentiator
SuitAI PRO unifies multiple AI capabilities in one product, reducing tool switching and keeping your work environment under local control.
